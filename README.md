# Ipv6-Configuration
Ce projet consiste à configurer et tester un réseau IPv6 en utilisant des équipements Cisco Systems (routeurs, switches). L'objectif est de comprendre l'adressage IPv6, le routage IPv6, le Neighbor Discovery Protocol (NDP) et le Dual-stack configuration.
# Architecture du réseau
## Routeur R1 (Cisco 2811)
- F0/0: 2001:DB8:1:1::1/64
- F1/0: 2001:DB8:1:2::1/64
- Link-local:FE80::1
